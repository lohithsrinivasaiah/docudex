# DocuDex

![Development Status](https://img.shields.io/badge/development-Active-brightgreen)
![Last Commit](https://img.shields.io/github/last-commit/lohithsrinivasaiah/docudex)
![Top Language](https://img.shields.io/github/languages/top/lohithsrinivasaiah/docudex)

**DocuDex** is a Python-based Text User Interface (TUI) tool that allows you to interact with documents using large language models (LLMs) directly from your terminal. It supports various document formats like PDF and DOCX, enabling seamless document processing and information retrieval.

## Features

- **Seamless Document Interaction**: Load and interact with documents using LLMs.
- **Multi-format Support**: Compatible with PDF, DOCX, and many other file formats.
- **Flexible Configuration**: Customize document processing and retrieval settings.
- **Extensible and Customizable**: Easily extendable to fit different use cases.

## Installation

### Steps


1. **Linux**
   You’re clearly someone who knows what they’re doing. Just type this:
   ```bash
   pip install docudex
   ```
   And boom, you're all set. No hand-holding required.

2. **Windows**
   Oh, you’re using Windows? Bold choice. Here’s the command anyway:
   ```bash
   pip install docudex
   ```
   Don’t worry, we won’t judge… much.

3. **macOS**
   Ah, a premium user. We see you. Treat yourself to a command:
   ```bash
   pip install docudex
   ```
   Your sleek, premium machine will handle the rest, because if it didn’t, it wouldn’t be worth the investment, right?

4. **Run DocuDex**
   To run DocuDex, use:
   ```bash
   docudex -h
   ```

5. **Version Check**
   To check the installed version of DocuDex, use:
   ```bash
   docudex -v
   ```

## Usage

To use DocuDex, simply pass a file as an argument to the CLI:

```bash
docudex -d path/to/your/document.pdf
```
